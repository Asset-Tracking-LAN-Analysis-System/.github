# ATLAS – Asset Tracking & LAN Analysis System

ATLAS is a tool designed to simplify asset inventory and network planning in technical environments.
It allows users to track, manage, and organize physical assets while also providing tools for LAN
structure planning and infrastructure documentation.

The system includes a web application that allows assets to be scanned and identified using
numeric codes directly from a mobile device, enabling fast and practical inventory workflows.

## Key Features

- Asset inventory management
- Structured documentation of hardware and equipment
- LAN infrastructure planning
- Mobile-friendly web interface for scanning asset codes
- Designed for technical labs, server environments, and IT infrastructure

ATLAS aims to provide a simple but powerful system for keeping track of devices, equipment,
and network structure in one place.
